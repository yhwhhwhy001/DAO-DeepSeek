# 修仙 Rougelike 设计规格

> **Status:** Draft | **Date:** 2026-06-08

## 1. 概述

**目标：** 在 DAO Genesis 宇宙中加入玩家角色——扮演一名修士 cell，WASD 实时操控、法术战斗、境界突破、轮回转世。

**模式：** 按 Tab 切换 HUD 游戏模式（镜头跟随、法术栏、状态浮层）和 观察模式（全景视角、熵/排行面板）。

---

## 2. 境界与突破

| 境界 | 灵力门槛 | 解锁 | 劫难概率 |
|------|---------|------|---------|
| 练气 | 0 | 吐纳术 | 0% |
| 筑基 | 10 | +护体罡气 | 10% |
| 金丹 | 30 | +神念探查 | 20% |
| 元婴 | 60 | +夺舍、功法槽+1 | 30% |
| 化神 | 100 | +丹药合成 | 40% |
| 渡劫 | 200 | +功法合成 | 50% |

**突破：** 灵力达标后自动触发。`random() > 劫难概率` → 成功。失败扣 50% 灵力。

**劫难表现：** 周围 3×3 灾难爆发（细胞随机死亡）。

---

## 3. 法术体系

```
键盘 1-7:

[1] 吐纳术    无消耗     吸收 5×5 残骸 (3x 效率)
[2] 护体罡气   5 灵力     10 tick 伤害减半
[3] 神念探查   3 灵力     7×7 区域显示灵力值 (5 tick)
[4] 血遁术    15 灵力     随机跳跃 15 格外空位
[5] 夺舍术    50 灵力     占据相邻 cell 为新身体

[6] 聚灵丹    1 天材地宝  回复 30% 灵力上限
[7] 悟道丹    2 天材地宝  20 tick Q-learning 速率 x3
```

天材地宝：energy > 5 的残骸，路过自动拾取。

---

## 4. 功法

Knowledge Engine 发现的知识可参悟为功法：

| 功法 | 条件 | 效果 |
|------|------|------|
| 金灵诀 | 含 type0 符号 | 吸收 type=0 残骸 +50% |
| 木灵诀 | 含 type1 符号 | 吸收 type=1 残骸 +50% |
| 水灵诀 | 含 type2 符号 | 吸收 type=2 残骸 +50% |
| 火灵诀 | 含 type3 符号 | 吸收 type=3 残骸 +50% |
| 混元诀 | 通用符号 | 全部 +20% |
| 遁法真解 | MOVE 相关 | 移动消耗 -50% |
| 不灭金身 | STAY 相关 | 伤害减免 +30% |

功法槽: 练气 1 / 筑基 2 / 元婴 3。轮回保留。

---

## 5. 轮回

```
死亡 → 轮回面板 (得分摘要)
      → 随机全图空位重生
      → 灵力 = 生前 30% + 5
      → 功法全部保留
      → 丹药清空
      → 境界重置为练气
```

## 7. 文件结构

```
src/
  cultivator.py            # 修士 PlayerCell + 境界 + 法术 + 突破

client/src/
  hooks/
    useKeyboard.ts         # WASD + 数字键监听
    useCultivator.ts       # 修士状态管理
  components/
    HUDOverlay.tsx          # HUD 顶部状态栏 + 底部法术栏
    SpellBar.tsx            # 法术按钮
    RealmIndicator.tsx      # 境界显示
    EventLog.tsx            # 事件日志
    ReincarnationModal.tsx  # 轮回弹窗
  engine/
    CameraController.ts     # 镜头跟随玩家
    PlayerRenderer.ts       # 玩家标记 (金色光晕)

server/main.py              # MODIFY: 玩家操作消息协议
```

## 8. WebSocket 协议扩展

```
客户端 → 服务端:
  {"type": "player_move", "dx": 1, "dy": 0}
  {"type": "player_spell", "spell": 3}
  {"type": "player_reincarnate"}

服务端 → 客户端:
  {"type": "player_state", "energy": 72, "realm": "金丹", ...}
  {"type": "player_died", "cause": "decay", "stats": {...}}
```

## 9. 不在本阶段范围

- 丹药炼制的复杂配方系统
- 多人同时在线
- 功法主动释放效果（均为被动 Buff）
- 天劫的多样化表现（默认只有灾难爆发一种）
- 成就系统
